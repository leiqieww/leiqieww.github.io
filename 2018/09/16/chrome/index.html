chrome缩略图的去除方法

发表于 2018-09-16

网上找了好久，总结了一下，chrome版本号 69.0.3497.92 ,实测可用。

首先在 chrome 导航栏中输入 chrome:version 查看自己的 chrome 版本号。

然后下载 ChromePak 工具，

https://shuax.com/portfolio/tools/

这里有两种工具，如果你的 chrome 版本是 62.0.3170.0 以后的就下载 “ChromePAK V5” ，之前的就下载 “ChromePak” 。

然后在 C / Program Files (x86) / Google / Chrome / Application / 69.0.3497.92 / 路径下找到 resources.pak 文件，复制它。

将 ChromePAK 工具解压后进入 pak_tools / windows / amd64 / 目录下，将 resources.pak 文件粘贴过来，在空白处同时按下 shift 和 鼠标右键，选择在此处打开 Powershell 窗口。

将 paktools.exe 拖入 Powershell 窗口 ，空格 输入命令 “-c=unpack -f=” ,再将 resources.pak 拖入窗口，回车运行。完成后会生成一个 resources 文件夹，在其中找到unknow文件夹，用编辑器打开298文件，找到most visited，在其中添加” display: none !important;” ,保存，退出。

然后再把 paktools.exe 拖入 Powershell 窗口中，空格，输入”-c=repack -f=” ,再把 resources.json 文件拖进去，回车运行。

最后把修改过的 resources.pak 文件放回原来的地方替换，大功告成！
