hexo+github搭建私人博客

发表于 2018-09-22

虽然博客时代已经过去，但是写博客仍然是一个非常好的锻炼自己的方式。

1：写博客可以记录自己技术成长的过程。

2：写博客可以总结和巩固自己的知识。

3：写博客可以锻炼自己的写作表达能力。、

4：写博客也可以积累自己的东西，能提升自己的价值并帮助他人。

而自己搭的博客可以说是真正属于自己的，没有任何束缚。

将自己的域名关联上 Github Pages 上（可选）

首先购买一个域名，比如腾讯云，阿里云什么的，也可以在 stormzhang 推荐的 godaddy.com 和「万网」上购买。

然后在 blog/source 文件夹下和 github 仓库里创建文件 CNAME （没有后缀名），编辑内容，填写申请的域名。

在申请域名的网站找到“域名解析”添加两条解析记录，

1：记录类型：CNAME ;主机记录：www ;记录值填自己的仓库地址（yourname.github.io）。

2：记录类型同1;主机记录：@ ；记录值同1.

等待大约10分钟后记录生效，直接输入你的域名就可以看到你的网站了！
